| --- | --- | --- |

## IVR フローをトリガーする

`POST /ivr/trigger`

IVR（自動音声応答）通話フローを開始し、開始されたセッションの `session_id` を返します。

<Warning>
  このエンドポイントは PRD（§2.1.5）に記載されていますが、リクエスト／レスポンスのスキーマはソース資料に**まだ定義されていません**。以下のフィールドは暫定です。実装前に Comdesk プラットフォームチームと契約内容を確認してください。
</Warning>

### レスポンス

<ResponseExample>

```json 200
{
  "data": {
    "session_id": "ivr_..."
  }
}
```

</ResponseExample>
